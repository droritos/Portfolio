UNREAL 5 / C++

Arena Shooter Prototype

Genre: First-Person Arena Shooter.
A technical showcase for gameplay programming, custom AI behavior, and modular component design in Unreal Engine 5.

Key Features

  • Custom Weapon System: Engineered a modular WeaponComponent in C++ utilizing Line-Tracing for accurate hit detection.
  • Dynamic Recoil: Implemented a procedural recoil system utilizing a custom FRecoilData struct for smooth weapon offset recovery.
  • Modular AI: Built enemy AI powered by Behavior Trees and Blackboards, featuring patrol routines and player detection logic.
  • Interface-Driven Damage: Created an IBulletHitInterface to handle interactions between projectiles and the world.
  • Animation Blending: Layered Blend Per Bone setup separating lower-body locomotion from upper-body weapon manipulation.

Technical Highlights

Component-Based Architecture

The project heavily utilizes Unreal's Actor Component system to keep classes lightweight and reusable (e.g., WeaponComponent handling socket attachment and firing logic).

Performance Optimization

To reduce overhead, frequently accessed variables like WeaponComponent references and socket transforms are cached at initialization rather than fetched on every tick.
