Turn10

Genre: Relaxing 1010-Style Mobile Puzzle.
Role: Lead Developer
Status: Prototype Phase
The Hook: Turn10 is a mobile-first puzzle game where you place geometric shapes on a grid to clear lines. Unlike traditional block puzzles, you can **rotate shapes**, adding a layer of deep strategy to a relaxing experience.

Unity C# Mobile UX DOTween

Core Gameplay

  • Shape Rotation: The core hook rotate shapes before placement to manage board space effectively.
  • Dynamic Grid System: A robust architecture for real-time shape interaction and line-clearing logic.
  • Tranquility Bonuses: Reward system for clearing multiple lines simultaneously, emphasizing set-up over speed.

Zen Experience

  • Droplet UI: Minimalist visual design focused on soft colors and clean translucency to reduce eye strain.
  • Satisfying Juice: Subtle DOTween animations and tactile sound effects crafted via AI-guided synthesis.
  • Pure Meditation: Focus on a singular Zen Mode with no timers, allowing for a focused, calming session.

Technical Foundation

Built in a rapid development environment leveraging AI-assisted workflows within the Antigravity IDE.

Data-Driven

ScriptableObject architecture for easy balance tweaks and shape additions.

Modern Input

Utilizes Unity's New Input System for precise mobile touch controls.

Modular Logic

Clean separation between Grid, Score, and Shape management systems.

Post-Mortem & Future Iterations

"A critical part of the process was learning when to cut features. Subtractive design—removing complex shaders and floating text—strengthened the core identity of the game as a relaxing experience."

• Next Step: Transitioning from Singleton patterns to an Event Bus for improved scalability.

• Lesson: AI-assisted scaffolding combined with manual Prefab refining is an incredibly efficient workflow.
